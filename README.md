# DigiTools Platform

Welcome to DigiTools! 🚀 এটি একটি modern digital tools marketplace যেখানে আপনি পাবেন premium AI tools, design assets, templates, এবং productivity software - সবকিছু এক জায়গায়!

## 🌟 কি কি পাবেন এখানে?

- **🛍️ Product Catalog**: আপনার পছন্দের digital tools খুঁজে নিন
- **🛒 Shopping Cart**: সহজেই product add করুন cart এ
- **📱 Responsive Design**: Mobile, tablet, desktop - সবখানেই perfect
- **🎨 Modern UI**: Tailwind CSS + DaisyUI দিয়ে সাজানো
- **🔔 Toast Notifications**: User-friendly notification system
- **🧩 Component-Based**: Clean এবং maintainable code

## 🛠️ কি কি টেকনোলজি ব্যবহার করা হয়েছে?

- **Frontend**: React 19 (সবচেয়ে latest version!)
- **Build Tool**: Vite (super fast!)
- **Styling**: Tailwind CSS + DaisyUI (beautiful এবং responsive)
- **Icons**: Lucide React (modern icons)
- **Notifications**: React Toastify (nice notifications)
- **Package Manager**: npm

## 📦 কিভাবে চালাবেন এই project?

খুবই সহজ! মাত্র ৩ ধাপে:

### ১ম ধাপ: Repository Clone করুন
```bash
git clone https://github.com/mdsadrulhasandider/B13-A6-DigiTools-Platform.git
cd B13-A6-DigiTools-Platform/DigiTools-Platform
```

### ২য় ধাপ: Dependencies Install করুন
```bash
npm install
```

### ৩য় ধাপ: Development Server চালু করুন
```bash
npm run dev
```

ব্যাস! এবার আপনার browser এ `http://localhost:5173` তে ভিজিট করুন! 🎉
